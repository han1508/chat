
# chat

A chatting app. Uses Socket.io for real time communication and stores user details in mySQL database

## Tech Stack

**Frontend:** React JS

**Server:** Node JS, Express JS

**Database:** Mongo DB
  

Go to the project directory

```bash
  cd chat
```

Install dependencies

```bash
  yarn install
```

```bash
  cd frontend/
  yarn install
```

Start the server

```bash
  yarn dev
```
Start the Client

```bash
  //open now terminal
  cd frontend
  yarn start
```
